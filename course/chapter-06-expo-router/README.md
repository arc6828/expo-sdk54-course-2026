# บทที่ 6: ระบบนำทางและสถาปัตยกรรมการเปลี่ยนหน้าจอด้วย Expo Router (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้ศึกษาคุณสมบัติที่พัฒนาและปรับปรุงมาอย่างยอดเยี่ยมใน Expo SDK 54 นั่นคือกลไกการนำหน้าจอแบบอิงพิกัดเส้นทางไฟล์ (File-based Routing) ด้วยเครื่องมือ **Expo Router v6** ซึ่งเพิ่มความสะดวดจัดสรรเลย์เอาต์หน้าจอซ้อนสะสม Stack, แถบนำทาง Tabs, การเชื่อมลิงก์โอนย้ายข้อมูลข้ามหน้า ตลอดจนการทำหน้ารายละเอียดไดนามิกดึงพารามิเตอร์มาประมวลผล

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม สื่อประกอบการสอน และหัวข้อการวัดประเมินทักษะจัดโครงสร้างเส้นทางนำหน้าจอ
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาหลักทางทฤษฎีอธิบายแนวคิด File-based Routing การสร้าง Layout คุมตำแหน่งด้วย `_layout.tsx` (Stack & Tabs layout), การเรียกใช้ `<Link>` และ `router.push()`/`replace()`/`back()`, การเขียนไฟล์ไดนามิก `[id].tsx` และดักอ่าน Query ด้วย `useLocalSearchParams`
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   แบบทดสอบปรนัยทบทวนความเข้าใจ 10 ข้อ ภาษาไทยเจาะลึกโครงสร้าง `app/` ความแตกต่างประวัติสแต็ค และการตั้ง Sticky Tabs พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้าง **แอปพลิเคชันแนะนำเมืองท่องเที่ยวระดับโลก (Travel Guide App)** เพื่อฝึกเขียนตั้งค่าเลย์เอาต์ Stack คุมโครงการ, วาดลิสต์ปุ่มเมือง, และเขียนหน้าเจาะลึกรับค่าไอดีดักเซ็ต Header บาร์ด้านบนตามชื่อเมืองที่กดจริง พร้อมโค้ดเฉลยฉบับสมบูรณ์ (Complete Solution)

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะได้ฝึกเขียนระบบนำทางแบบมาตรฐานอุตสาหกรรม:
* **Root Layout Stack:** ตั้งและจัดระเบียบบน `app/_layout.tsx` คุมสีแถบแสดงสถานะบาร์ส่วนหัวและเซ็ตสีปุ่ม Back
* **Travel Catalog Home:** หน้าสารบัญแสดงการ์ดภาพเมือง (โตเกียว, ปารีส, นิวยอร์ก) ด้วย `FlatList` และผูกคำสั่ง `router.push` ย้ายหน้า
* **Dynamic Route Details:** เขียนคุมเส้นทางที่ `app/destination/[id].tsx` ดักจับค่ารหัสเมืองท่องเที่ยว ID และอ่านคำบรรยายเฉพาะเจาะจง
* **Dynamic Header Bar:** ปรับข้อความตัวหนังสือหัวเรื่องบาร์ตาม cityName ที่ทรานสิทผ่านมา และมีปุ่ม `router.back()` เพื่อถอยหน้าจอ
