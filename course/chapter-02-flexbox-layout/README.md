# บทที่ 2: การจัดเลย์เอาต์หน้าจอด้วย Flexbox (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้เรียนรู้หัวข้อที่สำคัญที่สุดจุดหนึ่งในการสร้างอินเทอร์เฟซของแอปพลิเคชัน นั่นคือระบบการจัดแนวและจัดวางองค์ประกอบด้วย Flexbox ใน React Native ซึ่งนำเสนอความก้าวหน้าและการจัดการระยะที่ยืดหยุ่นต่างจาก CSS บนเว็บดั้งเดิม

---

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม วิธีการจัดกิจกรรม และมาตรการประเมินผลสัมฤทธิ์ในคาบเรียนเรื่องเลย์เอาต์
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาทฤษฎีอธิบายคุณสมบัติขนาด (Box Model), การใช้งาน `flexDirection`, การจัดกึ่งกลางชิดขอบด้วย `justifyContent`/`alignItems` และการใช้คุณสมบัติสเปซช่องไฟ `gap` ใน SDK ใหม่
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   คำถามปรนัยทบทวนความเข้าใจ 10 ข้อ เจาะลึกการจัดตำแหน่ง การเว้นระยะขอบ สัดส่วน `flex` และการคำนวณวงกลมมน พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการออกแบบหน้าจอ **แดชบอร์ดจำลองสมาร์ทโฮม (Smart Home Dashboard Mockup)** ทีละขั้นตอนโดยอาศัยคอมโพเนนต์ย่อยพื้นฐาน `View` และ `Text`

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะสร้างส่วนจัดวางเลย์เอาต์หน้าจอจำลอง 4 ส่วนเชื่อมต่อกัน:
* **Header Bar:** จัดทิศทางแนวนอนสไตล์ Space Between เพื่อแสดงชื่อแอปและสัญญานสถานะ Online
* **Status Card:** การ์ดสี Slate เข้มเพื่อจำลองการปรับแต่ง `padding`, `margin` และ `borderRadius`
* **2x2 Grid Menu:** จัดเรียงบล็อกสีทั้ง 4 บล็อกด้วย `flexWrap: 'wrap'` และคั่นกลางด้วย `gap: 16`
* **Footer Navigation Bar:** สร้างแท็บปุ่มจำลองลอยตัวชิดด้านล่างสุดโดยเกลี่ยพื้นที่ว่างเสมอกันด้วย `justifyContent: 'space-evenly'`
