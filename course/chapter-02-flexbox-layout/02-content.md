# บทที่ 2: การจัดเลย์เอาต์หน้าจอด้วย Flexbox (Content)

ระบบเลย์เอาต์ (Layout System) ของ React Native ทำงานโดยการอ้างอิงและประยุกต์ใช้ข้อกำหนดของ **Flexbox** เป็นหลัก ซึ่งมีแนวคิดคล้ายกับ CSS บนเว็บ แต่มีคุณสมบัติเฉพาะตัวบางประการที่ออกแบบมาเพื่อให้แสดงผลบนขนาดหน้าจอสมาร์ทโฟนที่มีสัดส่วนหลากหลายได้อย่างสม่ำเสมอ

---

## 2.1 การเขียนและประกาศสไตล์ใน React Native

ใน React Native เราไม่มีคลาสสไตล์ชีทแบบ CSS ไฟล์แยกภายนอก (เช่น `.css`) แต่จะใช้งานผ่านออบเจกต์ JavaScript ในรูปแบบ **CSS-in-JS** ซึ่งสามารถเขียนประกาศได้ 2 รูปแบบหลัก:

### 1. แบบ Inline Styles
เป็นการส่งออบเจกต์สไตล์เข้าไปยังพร็อพส์ `style` ของคอมโพเนนต์โดยตรง เหมาะสำหรับใช้กำหนดคุณสมบัติที่เปลี่ยนแปลงตามสถานะตัวแปร หรือทดสอบเลย์เอาต์แบบเร่งด่วน:
```tsx
<View style={{ width: 100, height: 100, backgroundColor: 'red' }} />
```

### 2. แบบ StyleSheet.create (แนะนำ)
เป็นการประกาศชุดสไตล์ภายนอกคอมโพเนนต์ ทำให้แยกส่วนการแสดงผลและสไตล์ชัดเจน ตัวคอมไพเลอร์ของ React Native สามารถแปลรหัสและแคชโครงสร้างสไตล์นี้ส่งไปยังฝั่งระบบปฏิบัติการได้มีประสิทธิภาพสูงกว่า:
```tsx
import { StyleSheet, View } from 'react-native';

export default function App() {
  return <View style={styles.box} />;
}

const styles = StyleSheet.create({
  box: {
    width: 100,
    height: 100,
    backgroundColor: '#38bdf8',
  },
});
```

## 2.2 โครงสร้างกล่องและคุณสมบัติขนาดขอบเขต (Box Model)

การกำหนดกล่องสไตล์ชีทประกอบด้วยคุณสมบัติสำคัญที่เป็นพื้นฐานในการจำกัดพื้นที่ ได้แก่:

* **`width` & `height`:** กำหนดความกว้างและความสูง สามารถใส่เป็นค่าพิกเซลเดี่ยว (เช่น `150`) หรือค่าเปอร์เซ็นต์ (เช่น `'50%'`)
* **`margin` (ระยะขอบภายนอก):** ระยะห่างระหว่างขอบกล่องภายนอกไปยังกล่องข้างเคียง
* **`padding` (ระยะขอบภายใน):** ระยะห่างระหว่างขอบกล่องด้านในไปยังวัตถุหรือเนื้อหาข้างในกล่อง
* **`border` (เส้นขอบ):** กำหนดขนาดและสีขอบด้วย `borderWidth`, `borderColor`
* **`borderRadius` (มุมโค้งมน):** ใช้ตัดมุมกล่องให้โค้งงอ (หากต้องการตัดกล่องให้กลมสนิท ให้กำหนด `borderRadius` เป็นครึ่งหนึ่งของขนาดความกว้าง/สูง)

```tsx
// ตัวอย่างกล่องโปรไฟล์ที่ใช้คุณสมบัติสเปซขอบ
const styles = StyleSheet.create({
  card: {
    width: 300,
    padding: 16,        // ระยะเว้นข้อความข้างใน
    margin: 8,          // ระยะเว้นนอกการ์ด
    borderWidth: 2,
    borderColor: '#e2e8f0',
    borderRadius: 12,   // ทำมุมการ์ดโค้งมนสไตล์การ์ดไอที
  }
});
```

## 2.3 ทิศทางการวางแกนด้วย flexDirection และ flex

หัวใจของการจัดตำแหน่งใน React Native คือทิศทางการวิ่งของข้อมูลในกลุ่มคอนเทนเนอร์ (Container):

### 1. คุณสมบัติ flexDirection (ทิศทางแกนหลัก)
กำหนดทิศทางเรียงตัวของคอมโพเนนต์ลูก (Children) ทั้งหมดภายในกล่องคอนเทนเนอร์นั้น:

| ค่าคงที่ | คำอธิบาย |
| :--- | :--- |
| **`column`** (ค่าเริ่มต้น) | เรียงตัวจาก **บนลงล่าง** (แกนหลักวิ่งแนวตั้ง, แกนรองวิ่งแนวนอน) |
| **`row`** | เรียงตัวจาก **ซ้ายไปขวา** (แกนหลักวิ่งแนวนอน, แกนรองวิ่งแนวตั้ง) |
| **`column-reverse`** | เรียงตัวจาก **ล่างขึ้นบน** ย้อนกลับตามแนวตั้ง |
| **`row-reverse`** | เรียงตัวจาก **ขวาไปซ้าย** ย้อนกลับตามแนวนอน |

> [!IMPORTANT]
> **ความแตกต่างสำคัญ:** บนเว็บบราวเซอร์ ค่าเริ่มต้นของ Flexbox จะเป็น `row` (แนวนอน) แต่ใน **React Native** ค่าเริ่มต้นถูกตั้งใจตั้งไว้เป็น `column` (แนวตั้ง) เพื่อสอดคล้องกับพฤติกรรมการเลื่อนหน้าจอสมาร์ทโฟน

### 2. คุณสมบัติ flex (สัดส่วนการยืดขยาย)
การสเกลพื้นที่ของแต่ละกล่อง หากกำหนด `flex: 1` หมายความว่าให้กล่องนั้นครอบครองพื้นที่ว่างทั้งหมดที่เหลืออยู่ในแถวนั้น

```tsx
// กล่องคอนเทนเนอร์ขนาดใหญ่คลุมหน้าจอ และแบ่งพื้นที่ออกเป็นสองบล็อกเท่ากัน
const styles = StyleSheet.create({
  container: {
    flex: 1, // ขยายครอบพื้นที่หน้าจอทั้งหมด
    flexDirection: 'row', // จัดวางแบบแนวนอน (ซ้ายไปขวา)
  },
  leftColumn: {
    flex: 1, // เอาพื้นที่ครึ่งหนึ่งทางซ้าย
    backgroundColor: '#1e293b',
  },
  rightColumn: {
    flex: 1, // เอาพื้นที่อีกครึ่งหนึ่งทางขวา
    backgroundColor: '#3b82f6',
  }
});
```

## 2.4 การจัดกึ่งกลางและการจัดเรียงด้วย justifyContent และ alignItems

เมื่อกำหนดทิศทางแกนแล้ว เราจะใช้คำสั่งอีก 2 ชุดในการระบุการกระจายตัวของวัตถุ:

### 1. justifyContent (จัดวางในทิศทางแกนหลัก)
* **`flex-start`:** ชิดต้นสายของแกนหลัก
* **`center`:** จัดวัตถุทั้งหมดให้อยู่ตรงกึ่งกลางของแกนหลัก
* **`flex-end`:** ชิดท้ายสายของแกนหลัก
* **`space-between`:** กระจายตัวบล็อกโดยเว้นระยะห่างตรงกลางให้กว้างที่สุดเท่ากัน (หัวท้ายติดขอบ)
* **`space-around`:** กระจายตัวโดยเว้นระยะรอบข้างบล็อกเท่า ๆ กัน (หัวท้ายมีช่องว่างครึ่งหนึ่งของตรงกลาง)
* **`space-evenly`:** กระจายช่องว่างระหว่างกล่องทุกบล็อกเท่ากันทั้งหมด รวมถึงหัวและท้าย

### 2. alignItems (จัดวางในทิศทางแกนรอง)
* **`stretch`** (ค่าเริ่มต้น): ยืดขยายบล็อกให้ยาวจนเต็มขอบแกนรอง (จะใช้งานได้เมื่อไม่มีการฟิกขนาดด้านนั้น)
* **`flex-start`:** ชิดจุดเริ่มต้นแกนรอง
* **`center`:** จัดวัตถุลูกทั้งหมดให้กึ่งกลางแนวแกนรอง
* **`flex-end`:** ชิดจุดสิ้นสุดแกนรอง

## 2.5 การขึ้นบรรทัดใหม่และการเว้นช่องว่าง (flexWrap & gap)

### 1. คุณสมบัติ flexWrap
ในกรณีที่วัตถุลูกมีความกว้างรวมกันเกินความกว้างคอนเทนเนอร์ หากตั้งค่า `flexWrap: 'nowrap'` (ค่าเริ่มต้น) กล่องจะล้นออกไปนอกหน้าจอหรือถูกบีบ แต่หากตั้งค่าเป็น **`wrap`** วัตถุที่ล้นจะถูกตัดไปขึ้นบรรทัดใหม่โดยอัตโนมัติ เหมาะสำหรับการจัดโครงสร้างแท็กสไตล์ หรือเมนู Grid ภาพไอคอน

### 2. คุณสมบัติ gap (ความสะดวกใน SDK 54)
ในอดีตการเว้นช่องไฟระหว่างบล็อกสีจะต้องเขียน `marginRight` หรือ `marginBottom` ให้วุ่นวาย แต่ใน Expo SDK 54 เราสามารถใช้คำสั่งในการสร้างระยะห่างได้อย่างเป็นระเบียบ:
* **`gap`:** ใส่ค่าพิกเซลเว้นระยะทั้งแนวตั้งและแนวนอนเท่า ๆ กัน (เช่น `gap: 12`)
* **`rowGap`:** เว้นระยะห่างเฉพาะแถวแนวตั้ง
* **`columnGap`:** เว้นระยะห่างเฉพาะคอลัมน์แนวนอน

## 2.6 โค้ดตัวอย่างหน้าเลย์เอาต์โครงสร้าง Grid จำลอง

ด้านล่างนี้คือโค้ดตัวอย่างการจัดตำแหน่งแผงปุ่มเมนู 4 บล็อก (Grid Layout 2x2) โดยประยุกต์ใช้ Flexbox และช่องไฟ `gap`:

```tsx
import { StyleSheet, Text, View } from 'react-native';

export default function GridLayout() {
  return (
    <View style={styles.container}>
      <Text style={styles.headerTitle}>เลย์เอาต์แบบตาราง 2x2</Text>
      
      <View style={styles.gridContainer}>
        {/* บล็อกที่ 1 */}
        <View style={[styles.box, { backgroundColor: '#ef4444' }]}>
          <Text style={styles.boxText}>เมนู A</Text>
        </View>
        {/* บล็อกที่ 2 */}
        <View style={[styles.box, { backgroundColor: '#3b82f6' }]}>
          <Text style={styles.boxText}>เมนู B</Text>
        </View>
        {/* บล็อกที่ 3 */}
        <View style={[styles.box, { backgroundColor: '#10b981' }]}>
          <Text style={styles.boxText}>เมนู C</Text>
        </View>
        {/* บล็อกที่ 4 */}
        <View style={[styles.box, { backgroundColor: '#f59e0b' }]}>
          <Text style={styles.boxText}>เมนู D</Text>
        </View>
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#0f172a',
    paddingTop: 60,
    paddingHorizontal: 20,
  },
  headerTitle: {
    fontSize: 22,
    fontWeight: 'bold',
    color: '#ffffff',
    textAlign: 'center',
    marginBottom: 20,
  },
  gridContainer: {
    flexDirection: 'row',     // วางแนวนอน
    flexWrap: 'wrap',         // ถ้าล้นให้ตัดขึ้นบรรทัดใหม่
    gap: 16,                  // เว้นระยะขอบรอบด้านทุกกล่อง
    justifyContent: 'center', // กระจายชิดใน
  },
  box: {
    width: '45%',             // กินความกว้างเกือบครึ่งหนึ่ง
    height: 120,
    borderRadius: 16,
    justifyContent: 'center', // จัดแกนหลักกึ่งกลาง (แนวตั้ง)
    alignItems: 'center',     // จัดแกนรองกึ่งกลาง (แนวนอน)
  },
  boxText: {
    fontSize: 18,
    fontWeight: 'bold',
    color: '#ffffff',
  },
});
```
