# บทที่ 9: พิกัดตำแหน่ง แผนที่ และเซนเซอร์อุปกรณ์ (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้ศึกษาเทคนิคระดับสูงในการจัดการข้อมูลทางภูมิศาสตร์และการโต้ตอบกับฮาร์ดแวร์เซนเซอร์เคลื่อนไหวของอุปกรณ์โทรศัพท์มือถือ โดยเน้นการจัดระบบตรวจสอบและขอรับอนุมัติสิทธิ์ (Runtime Location Permissions) ในการเข้าถึงพิกัด GPS, การถอดรหัสตำแหน่งละติจูดลองจิจูดให้ออกมาเป็นที่ตั้งทางกายภาพภาษาไทย (Reverse Geocoding) ด้วย `expo-location`, การวาดเรนเดอร์กราฟิกแผนที่และปักเข็มหมุดสัญลักษณ์ร้านค้าด้วย `react-native-maps` (`MapView` และ `Marker`) ตลอดจนการรับค่าข้อมูลชิปเซนเซอร์ความเร่ง `expo-sensors` (Accelerometer)

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม สื่อประกอบการสอน และหัวข้อการวัดประเมินทักษะจัดพิกัด แผนที่ และเซนเซอร์
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาหลักทางทฤษฎีอธิบายการจัดการ Runtime Permissions, ดึงพิกัดตำแหน่งอุปกรณ์, แปลงค่าที่อยู่ Reverse Geocoding, จัดแสดงแผนที่ด้วย MapView & Marker, คุมระยะซูม Delta, และการใช้งาน Accelerometer ติดสตรีม Cleanups คืนค่าแรมระบบ
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   แบบทดสอบปรนัยทบทวนความเข้าใจ 10 ข้อ ภาษาไทยเจาะลึกความแตกต่างสิทธิ์ Foreground/Background, ค่าพิกัด Delta ซูม, พฤติกรรมแกน Accelerometer, และระบบ Cleanup ป้องกัน Memory Leak พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้าง **ระบบแผนที่นำทางและสั่นสะเทือนจดจำพิกัด (Interactive Location Map Tracker)** เพื่อฝึกเขียนแอปแสดงพิกัดนำทาง ซูมภาพลึก ปักหมุด และผูกเซนเซอร์ความเร่งสั่นสะเทือนเครื่องเพื่อดีดมุมกล้องสไลด์กลับมาโฟกัสพิกัดผู้ใช้ปัจจุบันแบบเรียลไทม์ พร้อมโค้ดเฉลยฉบับสมบูรณ์ (Complete Solution)

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะได้ฝึกฝนการเขียนระบบพิกัดและเซนเซอร์เต็มรูปแบบ:
* **Location Permission Guard:** ร้องขอสิทธิ์ตำแหน่งเบื้องหน้า และสลับ UI แผงป้ายสีแดงเตือนพร้อมลิงก์ดิ่งไปหน้า Settings ของระบบปฏิบัติการเมื่อผู้ใช้ปฏิเสธสิทธิ์
* **Interactive Maps Rendering:** แสดงแผนที่พิกัดบ้านด้วย `react-native-maps` ซูมลึกระดับถนนปักเข็มหมุดสีสว่าง
* **Shake-to-Focus Sensors:** สั่นสะเทือนหรือเขย่าเครื่องเพื่อดีดกล้องหน้าจอแผนที่กลับมาศูนย์กลาง GPS ปัจจุบัน
* **Asynchronous Memory Cleanups:** ล้างปิดสตรีมเซนเซอร์คืนแรมเข้าสู่ระบบโทรศัพท์เมื่อคอมโพเนนต์อันเมานต์สำเร็จ
