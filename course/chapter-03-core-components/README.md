# บทที่ 3: คอมโพเนนต์พื้นฐานและการจัดการปฏิสัมพันธ์กับผู้ใช้ (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้เรียนรู้วิธีการทำงานกับคอมโพเนนต์พื้นฐาน (Core Components) ของ React Native ที่ทำหน้าที่เรนเดอร์องค์ประกอบบนหน้าจอจริง ได้แก่ รูปภาพ กล่องรับข้อความ พื้นที่เลื่อนจอ ตลอดจนการพัฒนา Custom Component และส่งผ่านตัวแปรข้ามส่วนข้อมูลด้วย Props ที่กำกับความถูกต้องดั้งเดิมผ่าน TypeScript Interface

---

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม สื่อประกอบการสอน และการกำหนดคะแนนการประเมินผลสัมฤทธิ์
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาทฤษฎีอธิบายการใช้งานคอมโพเนนต์ `Image` (รูปภาพเครื่องและเว็บ), `TextInput` (อินพุตข้อความและคีย์บอร์ด), `ScrollView` (พื้นที่เลื่อนจอ), `Pressable` (ปุ่มแบบโต้ตอบไดนามิก), `SafeAreaView` และแนวทาง Type-safety ของ Props ร่วมกับ TypeScript
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   คำถามปรนัยทบทวนความเข้าใจ 10 ข้อ วัดทักษะความรู้การเรียกใช้พร็อพส์เฉพาะเจาะจงของอินพุตและรูปภาพ ข้อแตกต่างของพื้นที่เลื่อน และการเขียน interface ตรวจสอบตัวแปร พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้าง **หน้าจอโปรไฟล์และฟอร์มตอบรับข้อมูลผู้ใช้งาน (Profile Form with Live Preview Card)** เพื่อให้ผู้เรียนออกแบบ Custom Component ที่เปลี่ยนค่าสเตตัสและหน้าโปรไฟล์ตามการป้อนค่าเรียลไทม์ได้จริง

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะได้ประมวลความรู้สร้างหน้าจอแก้ไขบันทึกประวัติผู้ใช้:
* **Live Profile Card (Custom Component):** ออกแบบคอมโพเนนต์การ์ดภายนอก รับข้อมูลผ่าน Props และแสดงรูปภาพวงกลมจาก URL ร่วมกับการใช้สเตตัสข้อความ
* **Interactive Inputs Form:** ฟอร์มรับข้อมูล (ชื่อ, ตำแหน่ง, รูปภาพ, ข้อความสเตตัส) พร้อมกำหนดรูปแบบแป้นพิมพ์ที่ตรงประเภทข้อมูล
* **Pressable Interaction:** ปุ่มกดแบบเนทีฟที่ลดสเกลขนาดกล่องลงชั่วขณะยามหน้าจอตรวจเจอการปัดหรือสัมผัสเพื่อส่งมอบการตอบสนองที่ลื่นไหล
