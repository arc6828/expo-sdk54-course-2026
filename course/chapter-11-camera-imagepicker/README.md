# บทที่ 11: การใช้งานกล้อง อัลบั้มภาพ และระบบจัดเก็บไฟล์เนทีฟร่วมกับคลาวด์ (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้ศึกษาเทคนิคเชิงปฏิบัติในการทำงานร่วมกับอินเทอร์เฟซฮาร์ดแวร์จริงของโทรศัพท์มือถือ โดยเน้นเรียนรู้เรื่องสิทธิ์ความปลอดภัยในอุปกรณ์เนทีฟ (Native Permissions), การเปิดพรีวิวเลนส์สดและถ่ายรูป Asynchronous ด้วยคอมโพเนนต์ `CameraView` ของ `expo-camera`, การเรียกบานหน้าต่างคลังรูปภาพมาแสดงผลสัดส่วนครอปจัตุรัสด้วย `expo-image-picker` และที่สำคัญคือความเข้าใจและการประยุกต์ใช้งานระบบจัดการไฟล์ 3 ระบบ ได้แก่ ไฟล์เฉพาะแอป (Private Sandbox via `expo-file-system`), ไฟล์แชร์ร่วมกับแอปอื่น (Shared Media via `expo-media-library`) และไฟล์บนคลาวด์ (Cloud Storage via `Supabase Storage`)

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม กิจกรรมชัตเตอร์จับภาพสด/กิจกรรมอัปโหลดไฟล์ผ่าน Cloud Network สื่อประกอบการเรียน และวิธีการวัดประเมินผลทักษะกล้องเนทีฟและระบบไฟล์ 3 ระดับ
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาหลักทางทฤษฎีอธิบายสิทธิ์กล้องและคลังรูปภาพ, การติดตั้งและใช้งาน `<CameraView>` ควบคุมพร็อพส์ `facing` สลับกล้องหน้าหลัง, เมธอดสั่งเซลฟี่ถ่ายภาพด้วย `takePictureAsync`, ตัวดึงภาพถ่ายแกลเลอรี `launchImageLibraryAsync`, การเปรียบเทียบและการเขียนสคริปต์ระบบจัดเก็บไฟล์ 3 รูปแบบ (Sandbox, Shared Storage, Cloud Storage)
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   แบบทดสอบปรนัยทบทวนความเข้าใจ 10 ข้อ ภาษาไทยเจาะลึกความรู้อุปกรณ์กล้องเนทีฟ ความแตกต่างของไดเรกทอรีจัดเก็บไฟล์ในแซนด์บ็อกซ์ การใช้สิทธิ์บันทึกเข้าคลังภาพเครื่องส่วนกลาง และวิธีการยิงอัปเดตไฟล์ขึ้นไปฝากที่ Supabase Storage พร้อมเฉลยและคำอธิบายโดยละเอียดครบถ้วนทุกตัวเลือก
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้างแอปพลิเคชัน **"จัดการรูปภาพโปรไฟล์อวาตาร์บันทึก 3 ระบบ (Profile Avatar Pro)"** เพื่อฝึกฝนสร้าง Modal กล้องถ่ายพรีวิวสด บันทึกรูปลง Sandbox (`documentDirectory`), บันทึกส่งแกลเลอรีส่วนกลาง (`expo-media-library`) และอัปโหลดขึ้นเซิร์ฟเวอร์คลาวด์ (`Supabase Storage`) พร้อมโค้ดเฉลยฉบับสมบูรณ์ (Complete Solution)

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะได้ฝึกฝนสร้างระบบกล้องและระบบไฟล์ระดับโปร:
* **Custom Live Camera View:** แสดงภาพพรีวิวเลนส์สด สลับหน้าเลนส์ และถ่ายรูปด่วน Asynchronous ด้วยการผูก References
* **Private Sandbox Engine:** คัดลอกภาพถ่ายจากโฟลเดอร์แคชชั่วคราวไปฝากไว้ใน `documentDirectory` แบบคงทน พร้อมผูกเก็บข้อมูลตัวชี้ใน `AsyncStorage` เพื่อกู้คืนภาพเมื่อเปิดแอปพลิเคชันใหม่
* **Shared Storage Export:** นำพาธรูปภาพมาเรียกสร้างสินทรัพย์ภาพ (Asset) ยิงบันทึกลงคลังแกลเลอรีหลักของเครื่องสมาร์ทโฟน ช่วยให้แอปภายนอกอื่นๆ ดึงรูปนี้ไปใช้ได้สำเร็จ
* **Supabase Storage Upload:** อ่านข้อมูลรูปภาพเป็นสตริง Base64 และถอดค่าเป็นไบนารี `ArrayBuffer` เพื่ออัปโหลดขึ้นคลาวด์บัคเก็ต ดึง Public URL มาบันทึกใช้งานแบบออนไลน์เรียลไทม์
