# บทที่ 5: การแสดงผลรายการข้อมูลและประสิทธิภาพการทำงาน (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้เรียนรู้เทคนิคการแสดงผลข้อมูลแบบรายการปริมาณมาก ซึ่งถือเป็นแกนหลักของการสร้างหน้าแอปพลิเคชัน Feed ต่าง ๆ โดยเน้นทำความเข้าใจสถาปัตยกรรม Virtualization และระบบหมุนเวียนกล่องคืนสภาพหน่วยความจำ (View Recycling) เพื่อพัฒนาการเรนเดอร์ที่มีประสิทธิภาพ ปราศจากปัญหาอาการเฟรมเรตตกหรือแรมเต็ม พร้อมผูกเชื่อมโยงข้อมูลจาก REST API ภายนอกด้วย Fetch API และมีระบบหมุนดึงโหลดรีเฟรช Pull-to-Refresh กับ Infinite Scroll

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อ วัตถุประสงค์เชิงพฤติกรรม และเกณฑ์วัดทักษะประสิทธิภาพเรนเดอร์ลิสต์และการดึงข้อมูลเน็ตเวิร์ก
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาทฤษฎีอธิบายละเอียดข้อจำกัดของ `ScrollView`, หลักทำงานหลังบ้านของ `FlatList`, พร็อพส์หลัก (`data`, `renderItem`, `keyExtractor`), ตัวอย่าง Fetch API โหลดข้อมูล REST API และการใช้งาน `SectionList` แยกกลุ่ม Sticky Headers
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   แบบทดสอบปรนัยทวนความเข้าใจ 10 ข้อ ในภาษาไทย เจาะลึกกลไกลิสต์ การผูกดักจับพารามิเตอร์ และการแยกแยะกรณี Sticky headers พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้าง **แอปพลิเคชันคลังแกลเลอรีภาพออนไลน์ (Live Photo Gallery Stream)** ดึงข้อมูลจริงจาก REST API แสดงผลด้วย `FlatList` ร่วมกับการทำ Infinite Scroll และ Pull-to-Refresh โดยแนบโค้ดเฉลยฉบับสมบูรณ์ (Complete Solution)

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะเขียนระบบสตรีมภาพออนไลน์:
* **Dynamic Fetching:** เขียนคุม `useEffect` ยิงดักจับโหลดข้อมูลจาก JSONPlaceholder Photos API ทีละ 10 รูปมาเซ็ตสเตต
* **Pull-to-Refresh:** ใช้เหตุการณ์ดึงหน้าต่างลงของ `onRefresh` เพื่อเคลียร์รายการกลับเป็นหน้าแรกและดึงรูปใหม่ล่าสุดมาทดแทน
* **Infinite Scroll (Pagination):** ใช้ `onEndReached` ช่วยสลับเพิ่มหน้าทีละหน้าและโหลดรูปเซ็ตถัดไปมารวมต่อท้ายรายการเก่าอัตโนมัติเมื่อผู้ใช้เลื่อนลงสุดจอ
* **List Decoration Elements:** ตกแต่งหัวข่าว ยอดจำนวนภาพ คั่นแยกการ์ดภาพด้วย Separator และโชว์กล่องข้อความเมื่อคลังข้อมูลเป็นศูนย์
