# บทที่ 1: ความรู้เบื้องต้นเกี่ยวกับ React Native Expo และการเตรียมสภาพแวดล้อมการพัฒนาแอปพลิเคชัน (Introduction & Environment Setup)

## หัวข้อเนื้อหาประจำบท
- ทำความเข้าใจสถาปัตยกรรมและข้อดีของ React Native และ Expo SDK 54
- การเปรียบเทียบความแตกต่างระหว่าง Expo Go, Development Builds และ React Native CLI
- การเตรียมความพร้อมและติดตั้งสภาพแวดล้อม (Node.js, VS Code, Git)
- การสร้างโปรเจคใหม่ด้วยคำสั่ง `create-expo-app` เวอร์ชันล่าสุด
- การล้างตัวอย่างโครงสร้างเดิมเพื่อเริ่มต้นพัฒนาด้วยคำสั่ง `npm run reset-project`
- โครงสร้างและส่วนประกอบต่าง ๆ ของโปรเจค React Native Expo (เช่น `package.json`, `app.json`, โฟลเดอร์ `app/`)
- การรันโปรเจคและทดสอบแอปพลิเคชันผ่าน Expo Go บนอุปกรณ์จริง และการจำลองระบบ (Android Emulator / iOS Simulator)

## วัตถุประสงค์เชิงพฤติกรรม
1. ผู้เรียนสามารถอธิบายสถาปัตยกรรม ความแตกต่าง และการเลือกใช้งานเครื่องมือระหว่าง Expo Go และ Development Builds ได้อย่างถูกต้อง
2. ผู้เรียนสามารถเตรียมความพร้อม ติดตั้งโปรแกรมพื้นฐาน และกำหนดค่าสภาพแวดล้อมสำหรับการพัฒนาแอปพลิเคชันได้สำเร็จโดยไม่มีข้อผิดพลาด
3. ผู้เรียนสามารถใช้คำสั่งสร้างโปรเจค React Native Expo SDK 54 และทำความเข้าใจเกี่ยวกับบทบาทของไฟล์การตั้งค่าที่สำคัญได้อย่างถูกต้อง
4. ผู้เรียนสามารถรันแอปพลิเคชันแรก (Hello World) และแสดงผลบนเครื่องมือทดสอบ เช่น Expo Go หรือ Simulator ได้อย่างสมบูรณ์

## วิธีสอนและกิจกรรมการเรียนการสอนประจำบท
1. **การบรรยายประกอบสไลด์ (Lecture):** แนะนำเนื้อหาทฤษฎี ความเป็นมาของ React Native ข้อดีของ Expo SDK 54 ตลอดจนเปรียบเทียบความคุ้มค่าของการพัฒนาในรูปแบบต่าง ๆ
2. **การสาธิตแบบมีส่วนร่วม (Interactive Demonstration):** ผู้สอนสาธิตขั้นตอนการติดตั้งซอฟต์แวร์ที่จำเป็นทั้งหมดทีละขั้นตอน และเปิดโอกาสให้ผู้เรียนติดตั้งตามไปพร้อม ๆ กัน
3. **การลงมือปฏิบัติ (Hands-on Practice):** ผู้เรียนปฏิบัติตามคู่มือการสร้างโปรเจคใหม่ และทดสอบการแก้ไขโค้ดเบื้องต้นเพื่อสังเกตผลกระทบแบบ Fast Refresh
4. **การระดมสมองและถามตอบ (Q&A & Brainstorming):** เปิดโอกาสให้ผู้เรียนสอบถามปัญหาที่พบขณะรันแอปพลิเคชัน และร่วมกันวิเคราะห์วิธีแก้ไขข้อผิดพลาดของการตั้งค่าเบลื้องต้น

## สื่อการเรียนการสอน
1. เอกสารสไลด์นำเสนอเรื่อง "Introduction to React Native Expo SDK 54"
2. คู่มือขั้นตอนการติดตั้งซอฟต์แวร์และเครื่องมือจำลองระบบปฏิบัติการ (Android Studio & Xcode)
3. วิดีโอสาธิตการทำงานเบื้องต้นและการรันแอปพลิเคชันผ่าน Expo Go บนสมาร์ทโฟน
4. โปรเจคตัวอย่างเริ่มต้นที่จัดเตรียมไว้บนแพลตฟอร์ม Git Repository

## การวัดผลและการประเมินผล

### 1. การวัดผล
1.1 วัดความรู้ความเข้าใจจากความถูกต้องในการตอบคำถามท้ายบทเรียน (Multiple Choice Quiz)
1.2 วัดผลสัมฤทธิ์การปฏิบัติงานจากการสร้างโปรเจคและการนำเสนอหน้าจอแอปพลิเคชันแรกบนอุปกรณ์จริง/ตัวจำลองในชั่วโมงเรียน
1.3 ตรวจสอบความถูกต้องของการแก้ไขและการบันทึกไฟล์โค้ดเริ่มต้นเพื่อแสดงผลข้อความตามโจทย์กำหนด

### 2. การประเมินผล
2.1 ผู้เรียนทำคะแนนแบบทดสอบปรนัยท้ายบทเรียนได้ไม่น้อยกว่าร้อยละ 80 (อย่างน้อย 4 จาก 5 ข้อ)
2.2 ผู้เรียนสามารถรันโปรเจคเริ่มต้นและแสดงผลหน้าจอ "Hello World" บนอุปกรณ์ทดสอบได้ผ่านเกณฑ์ (ผ่าน/ไม่ผ่าน)
2.3 การสังเกตพฤติกรรมและการมีส่วนร่วมในกิจกรรมตอบคำถามและการลงมือแก้ปัญหาในห้องเรียน
