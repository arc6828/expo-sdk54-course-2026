# บทที่ 1: ความรู้เบื้องต้นเกี่ยวกับ React Native Expo และการเตรียมสภาพแวดล้อม (Overview)

ยินดีต้อนรับสู่บทเรียนแรกของการพัฒนาแอปพลิเคชันมือถือ! ในบทเรียนนี้ ผู้เรียนจะได้ทำความเข้าใจเกี่ยวกับการเขียนแอปพลิเคชันระบบ Cross-Platform ด้วยเฟรมเวิร์ก React Native และเครื่องมือสร้างความสะดวกอย่าง Expo SDK 54 ตลอดจนลงมือติดตั้งสภาพแวดล้อมและรันระบบทดสอบแรก

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   รายละเอียดวัตถุประสงค์เชิงพฤติกรรม สื่อการเรียนรู้ ขั้นตอนแผนการจัดกิจกรรม รวมถึงมาตรการประเมินและวัดผลในชั่วโมงเรียน
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาทฤษฎีเปรียบเทียบสถาปัตยกรรม, แนวทางรันแอปประเภทต่าง ๆ, ขั้นตอนติดตั้ง Node.js/VS Code และการแนะนำโครงสร้างโฟลเดอร์
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   คำถามปรนัยทบทวนความเข้าใจจำนวน 10 ข้อ พร้อมตัวเลือกคำตอบและคำเฉลยอธิบายประกอบเหตุผลเบื้องหลังโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์และขั้นตอนการปฏิบัติงานสร้างแอปต้อนรับธีม Slate เข้ม (Dark Mode) สำหรับสแกนรันบนอุปกรณ์จริงผ่านแอปพลิเคชัน Expo Go

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในส่วนของการฝึกฝนทักษะท้ายบท ผู้เรียนจะได้ลงมือพัฒนาแอปพลิเคชันต้อนรับผู้ใช้งานเบื้องต้น โดยมีคุณลักษณะดังนี้:
* การติดตั้ง Node.js และโปรแกรมสร้างโค้ด VS Code
* การสร้างโปรเจคด้วยคำสั่งสร้างแอปเวอร์ชันล่าสุด
* การล้างโครงสร้างตัวอย่างด้วยคำสั่ง `npm run reset-project` เพื่อเริ่มต้นแอปพลิเคชันจากศูนย์
* การสร้างคอมโพเนนต์การ์ดข้อมูลส่วนตัวโดยประยุกต์ใช้คอมโพเนนต์พื้นฐาน `View`, `Text` และสไตล์จาก `StyleSheet`
* การทดสอบคุณสมบัติ Fast Refresh เพื่ออัปเดตหน้าตา UI ในโทรศัพท์มือถือขณะบันทึกโค้ดแบบเรียลไทม์
