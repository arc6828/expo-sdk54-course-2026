# บทที่ 7: การตกแต่งอินเทอร์เฟซ ไอคอน และรูปภาพประสิทธิภาพสูง (Lesson Plan)

## หัวข้อเนื้อหาประจำบท
- ระบบพิกัดตำแหน่งสัมพัทธ์และสัมบูรณ์ (`position: 'relative'` และ `'absolute'`) สำหรับวางองค์ประกอบแบบอิสระ
- การดาวน์โหลดและติดตั้งชุดแบบอักษรพิเศษ (Custom Fonts) จาก Google Fonts ผ่านระบบของ `expo-font` และการซ่อนหน้าจอ Splash Screen ชั่วขณะระหว่างรอโหลด
- การประยุกต์ใช้ชุดไอคอนประเภทเวกเตอร์ประสิทธิภาพสูง (Vector Icons) ด้วยการเลือกดึงไลบรารี `@expo/vector-icons`
- การเพิ่มประสิทธิภาพการจัดแสดงรูปภาพแอปพลิเคชันด้วยไลบรารี `expo-image` แทนคอมโพเนนต์ Image ดั้งเดิม
- ข้อดีของการใช้ `expo-image` เช่น การแคชภาพอัตโนมัติ (Advanced Caching), การแสดงผลแบบเลือนลางประคองสายตา (Blurhash Placeholders) และเอฟเฟกต์ Fade-in Transition
- การดักเช็คค่าสีหน้าจอระบบปฏิบัติการและประยุกต์ใช้งานโหมดมืด (Dark Mode / Theme Support) ด้วย `useColorScheme` Hook

## วัตถุประสงค์เชิงพฤติกรรม
1. ผู้เรียนสามารถระบุความแตกต่างและใช้คำสั่ง `position: 'absolute'` ในการวางชิ้นงานทับซ้อนไอคอนได้ถูกต้อง
2. ผู้เรียนสามารถเขียนโค้ดดาวน์โหลด Custom Fonts จาก Google Fonts มาใช้ปรับแต่งแบบอักษรในแอปได้สำเร็จ
3. ผู้เรียนสามารถเรียกใช้ชุดไอคอนจากระบบ `@expo/vector-icons` ในการเพิ่มความสวยงามให้ปุ่มเมนูนำทางได้สำเร็จ
4. ผู้เรียนสามารถเปลี่ยนโค้ดมาใช้ `expo-image` พร้อมตั้งค่าพร็อพส์ `placeholder` แบบ Blurhash ได้อย่างถูกต้อง
5. ผู้เรียนสามารถเขียนควบคุมการเปลี่ยนสไตล์สีสันแอปให้เปลี่ยนสลับสีตามโหมดมืด (Dark Mode) ของระบบโทรศัพท์ปลายทางได้สำเร็จ

## วิธีสอนและกิจกรรมการเรียนการสอนประจำบท
1. **การบรรยายและเปรียบเทียบเชิงวิเคราะห์ประสิทธิภาพ (Image Rendering Performance Lecture):** บรรยายการโหลดภาพแบบเดิมของแอปที่ทำให้กระตุก และนำเสนอเปรียบเทียบคุณประโยชน์ของ `expo-image` ในการเซฟแคชและใช้ระบบ Blurhash
2. **การทำ Live Coding Demonstration หัวข้อ Modern Profile Dashboard with Theme Switch:** ผู้สอนสาธิตการออกแบบหน้าระบบโปรไฟล์ระดับพรีเมียม (Premium Card Mockup) ที่เรียกใช้ฟอนต์ Outfit จาก Google Fonts, ใช้ไอคอนตกแต่งขอบมุมปุ่มด้วย Absolute position และใช้ระบบ `useColorScheme` สลับธีมสว่าง/มืด
3. **กิจกรรมแกะสลักสไตล์ดีไซน์ (Blurhash & Caching Challenge):** ให้ผู้เรียนทดลองดึงรูปจากเน็ตเวิร์กที่มีขนาดใหญ่มาก ๆ มาใส่ `expo-image` พร้อมปรับเปลี่ยนเอฟเฟกต์การ Fade ภาพตอนแสดงผล และสร้าง Blurhash สรุปหน้าตาการหมุน
4. **กิจกรรมร่วมมือรีแฟกเตอร์ฟอนต์ระบบ (Group Fonts & Icons Integration):** ให้กลุ่มผู้เรียนช่วยกันปรับปรุงโค้ดโปรเจกต์เดิมให้ไม่มีฟอนต์ดั้งเดิมของระบบ โดยต้องเปลี่ยนมาใช้ฟอนต์พิเศษทั้งหมดและดักจับไม่ให้แอปแสดงผลตัวหนังสือพังตอนติดตั้งหน้าแรกสุด

## สื่อการเรียนการสอน
1. เอกสารสไลด์เรื่อง "Advanced Styling, Custom Fonts, Vector Icons, and Expo Image in React Native"
2. ตัวอย่างสตริงรหัส Blurhash สั้นสำหรับใช้จำลองสกินก่อนโหลดภาพจริง
3. แหล่งข้อมูลสำหรับค้นหาไอคอนเวกเตอร์อย่างเป็นทางการของ Expo (icons.expo.fyi)
4. โครงการซอร์สโค้ดแดชบอร์ดตกแต่งความงามตั้งต้น (Style Sandbox Project) บนคลัง Git

## การวัดผลและการประเมินผล

### 1. การวัดผล
1.1 ประเมินระดับความรู้เชิงทฤษฎีจากการทำคะแนนแบบทดสอบปรนัยท้ายบทเรียน 10 ข้อ
1.2 ตรวจสอบโครงสร้างคำสั่งการเรียกใช้ฟอนต์ การป้องกันตัวหนังสือกระตุก และความสมบูรณ์ในการเรียกใช้เวกเตอร์ไอคอน
1.3 ตรวจประเมินทักษะการประยุกต์ใช้งานคอมโพเนนต์ `expo-image` และการเขียนระบบ Dark Mode ไดนามิกสไตล์ในปฏิบัติการ

### 2. การประเมินผล
2.1 ผู้เรียนทำคะแนนแบบทดสอบปรนัยท้ายบทเรียนได้ไม่น้อยกว่าร้อยละ 80 (อย่างน้อย 8 จาก 10 ข้อ)
2.2 ผู้เรียนสามารถสร้างหน้าแดชบอร์ดความงามที่เปลี่ยนโหมดสีมืดสว่างตามระบบเครื่อง และรันภาพผ่าน `expo-image` ที่มีตัวช่วยพรีวิวภาพเบลอได้สำเร็จผ่านเกณฑ์ (ผ่าน/ไม่ผ่าน)
2.3 สังเกตพฤติกรรมการแลกเปลี่ยนไอเดียศิลปะ ความตั้งใจประณีตในการเขียนระบุ CSS สไตล์ให้ตรงตามต้นแบบจำลอง
