# บทที่ 7: การตกแต่งอินเทอร์เฟซ ไอคอน และรูปภาพประสิทธิภาพสูง (Overview)

ในบทเรียนนี้ ผู้เรียนจะได้ศึกษาเทคนิคระดับสูงในการควบคุมความงามและการจัดวางองค์ประกอบอินเทอร์เฟซของแอปพลิเคชันให้ดูน่าดึงดูดและทำงานได้รวดเร็วตามมาตรฐานสากล โดยเน้นการจัดตำแหน่งทับซ้อนด้วย `position: 'absolute'`, การปรับแต่งชุดอักษรพิเศษผ่าน `expo-font`, การตกแต่งไอคอนประหยัดขนาดไฟล์ด้วย `@expo/vector-icons`, การเพิ่มระดับสตรีมความเร็วและประหยัดอินเทอร์เน็ตในการประมวลภาพผ่าน `expo-image` และพัฒนาโครงสร้างแอปพลิเคชันที่รองรับโหมดมืด (Dark Mode / Theme Support)

## 📂 สารบัญเอกสารประจำบทเรียน (Chapter Documents)
คุณสามารถเลือกอ่านและเข้าถึงไฟล์เอกสารต่าง ๆ ในโฟลเดอร์บทเรียนนี้ได้โดยตรง:

1. 📋 **[แผนการสอนประจำบท (01-lesson-plan.md)](./01-lesson-plan.md)**  
   หัวข้อการสอน วัตถุประสงค์เชิงพฤติกรรม สื่อประกอบการสอน และหัวข้อการวัดประเมินทักษะจัดโครงสร้างหน้าตาแอปพลิเคชัน
2. 📖 **[เนื้อหาหลักการเรียนรู้ (02-content.md)](./02-content.md)**  
   เนื้อหาหลักทางทฤษฎีอธิบายการใช้พิกัดตำแหน่ง `relative` และ `absolute`, การโหลด Custom Fonts ผ่าน `expo-font` ร่วมกับ `expo-splash-screen`, การดึงสัญลักษณ์ไอคอนจาก `@expo/vector-icons`, การประหยัดแบนด์วิดท์เซฟแคชรูปภาพด้วย `expo-image` (Blurhash / Transitions) และการสลับชุดสีตามธีมด้วย `useColorScheme`
3. 📝 **[แบบทดสอบท้ายบทเรียน (03-quiz.md)](./03-quiz.md)**  
   แบบทดสอบปรนัยทบทวนความเข้าใจ 10 ข้อ ภาษาไทยครอบคลุมพฤติกรรมของ Absolute Positioning, ระบบจัดสรร Fonts, ไอคอนเวกเตอร์, ประสิทธิภาพ Caching ของภาพพรีวิว และ Hook ตรวจจับโหมดสีตัวเครื่อง พร้อมเฉลยและคำอธิบายโดยละเอียด
4. 💻 **[คู่มือปฏิบัติการขั้นพื้นฐาน (04-lab.md)](./04-lab.md)**  
   โจทย์ปฏิบัติการสร้าง **การ์ดโปรไฟล์แดชบอร์ดระดับพรีเมียม (Premium Profile Card Dashboard)** เพื่อฝึกฝนเขียนควบคุมสถานะ Badge ออนไลน์ทับซ้อนมุมขวา, โหลดฟอนต์ Prompt ดักไม่ให้หน้าจอสัญลักษณ์เด้งพัง, ใช้รูปภาพความเร็วสูงที่มีสีเบลอรอโหลด และเขียนสลับเฉดสีพื้นหลังแอปเมื่อผู้ใช้เปลี่ยนโหมดหน้าจอโทรศัพท์เป็นโหมดมืด พร้อมโค้ดเฉลยฉบับสมบูรณ์ (Complete Solution)

## 🛠️ สรุปโปรเจคปฏิบัติการท้ายบทเรียน (Lab Project Summary)
ในปฏิบัติการของบทนี้ ผู้เรียนจะได้ฝึกฝนการเขียนดีไซน์โครงสร้างอินเทอร์เฟซมาตรฐานระดับพรีเมียม:
* **Custom Typography Loading:** เขียนคุม `useFonts` และเชื่อม Splash Screen ปิดหน้าหลักเมื่อโหลด Prompt เสร็จ
* **Absolute Overlay Positioning:** วางตำแหน่งจุด Badge สีเขียวนีออนตัดขอบขาวทับมุมขวาล่างของรูปภาพหลักของโปรไฟล์
* **High Performance Avatar Render:** ประมวลผลรูปภาพโปรไฟล์ผ่านคอมโพเนนต์ `expo-image` ร่วมกับเอฟเฟกต์ Fade-in และเฉดสี Blurhash
* **Adaptive Theme Colors:** ออกแบบและจัดคู่สีที่เปลี่ยนสไตล์ข้อความ ขอบ และพื้นหลังแอปโดยอัตโนมัติอิงตามการเปลี่ยนระบบเครื่องหลักของผู้ใช้งานระหว่าง Light และ Dark Mode
